# 战队管理系统使用指南

## 1. 项目概述

### 1.1 系统简介

战队管理系统是一个专为游戏战队设计的一站式管理平台，提供从成员管理、数据统计到智能分析的全方位功能。系统支持多种游戏类型，帮助战队提高管理效率，提升团队凝聚力。

### 1.2 核心价值

- **提升管理效率**：自动化管理流程，减少人工操作
- **数据驱动决策**：通过数据统计，优化战队策略
- **增强团队凝聚力**：提供沟通平台，促进队员交流
- **智能辅助决策**：AI 助手提供智能分析和建议

## 2. 系统架构

### 2.1 技术栈

- **前端**：Next.js 14、React 18、Tailwind CSS、Framer Motion
- **后端**：Supabase（数据库、认证、存储）
- **其他**：Chart.js（数据可视化）、Zhipu AI（智能助手）

## 4. 队长操作指南

### 4.1 战队创建与管理

#### 创建战队
1. 登录系统后，点击导航栏中的「战队」
2. 点击「创建战队」按钮
3. 填写战队名称、游戏类型、服务器等信息
4. 上传战队队标
5. 点击「创建」按钮完成创建

#### 管理队员
1. 进入战队管理页面
2. 点击「成员管理」选项
3. **邀请队员**：输入队员邮箱或游戏 ID 发送邀请
4. **审批申请**：查看并审批加入申请
5. **设置角色**：为队员设置队长、副队长等角色
6. **踢出队员**：移除不符合要求的队员

#### 发布招募
1. 进入战队管理页面
2. 点击「发布招募」选项
3. 填写招募条件、职位需求等信息
4. 点击「发布」按钮

### 4.2 数据管理

#### 比赛记录管理
1. 进入战队数据中心
2. 点击「比赛记录」选项
3. **添加记录**：填写比赛信息、对手、结果等
4. **编辑记录**：修改已有的比赛记录
5. **删除记录**：删除不需要的比赛记录

#### 训练计划管理
1. 进入战队数据中心
2. 点击「训练计划」选项
3. **创建计划**：填写训练内容、时间、参与人员
4. **分配任务**：为队员分配具体训练任务
5. **跟踪进度**：查看训练计划的执行情况

### 4.3 智能功能使用

#### 智能分组
1. 进入战队智能分组页面
2. 选择需要分组的队员
3. 点击「开始分组」按钮
4. 查看分组结果，可手动调整
5. 确认分组并保存

#### AI 助手
1. 进入战队 AI 聊天页面
2. 输入问题或需求
3. 等待 AI 助手的回答和建议
4. 可以继续追问或提出新的问题

### 4.4 交易管理

#### 发布商品
1. 进入战队出售页面
2. 点击「发布商品」按钮
3. 选择商品类型（战队/ID）
4. 填写价格、描述、联系方式等信息
5. 点击「发布」按钮

#### 管理商品
1. 进入战队出售管理页面
2. 查看已发布的商品
3. **编辑商品**：修改商品信息
4. **下架商品**：移除已出售或不再出售的商品

## 5. 队员操作指南

### 5.1 加入战队

#### 申请加入
1. 浏览招募大厅
2. 找到感兴趣的战队
3. 点击「申请加入」按钮
4. 填写申请信息和特长
5. 等待队长审批

#### 接受邀请
1. 收到战队邀请通知
2. 点击通知中的「查看邀请」
3. 查看战队信息
4. 点击「接受邀请」加入战队

### 5.2 个人中心

#### 个人资料管理
1. 点击导航栏中的「个人中心」
2. 编辑个人信息、游戏 ID、段位等
3. 上传个人头像
4. 保存修改

#### 查看战队信息
1. 进入战队空间
2. 查看战队基本信息
3. 查看队员列表
4. 查看战队数据统计

### 5.3 参与活动

#### 参加训练
1. 查看训练计划
2. 确认分配给自己的训练任务
3. 完成训练后标记为完成

#### 参加比赛
1. 查看比赛安排
2. 确认参赛状态
3. 比赛结束后查看结果和数据

### 5.4 社区互动

#### 发布帖子
1. 进入贴吧页面
2. 点击「发布帖子」按钮
3. 填写标题、内容、标签
4. 点击「发布」按钮

#### 评论互动
1. 浏览帖子
2. 在评论区输入评论
3. 点击「发布评论」按钮
4. 可以回复其他评论

### 5.5 好友管理

#### 添加好友
1. 进入好友页面
2. 点击「添加好友」按钮
3. 输入对方游戏 ID 或用户 ID
4. 发送好友请求

#### 管理好友
1. 查看好友列表
2. 点击好友头像查看详情
3. 可以删除好友或发送消息

## 6. 常见问题解答

### 6.1 登录问题

**Q: 忘记密码怎么办？**
A: 点击登录页面的「忘记密码」链接，按照提示重置密码。

**Q: 账号被锁定怎么办？**
A: 联系系统管理员解锁账号，或等待自动解锁（通常 24 小时后）。

### 6.2 战队管理问题

**Q: 如何转让队长？**
A: 进入战队管理页面，选择要转让的队员，点击「转让队长」按钮。

**Q: 如何修改战队信息？**
A: 进入战队管理页面，点击「编辑战队信息」按钮进行修改。

### 6.3 数据问题

**Q: 比赛记录添加错误怎么办？**
A: 进入比赛记录页面，找到错误记录，点击「编辑」按钮进行修改。

**Q: 数据统计不准确怎么办？**
A: 检查比赛记录是否完整，或联系系统管理员进行数据修复。

### 6.4 智能分组问题

**Q: 智能分组结果不满意怎么办？**
A: 可以手动调整分组结果，然后保存修改。

**Q: 为什么有些队员没有被分组？**
A: 可能是因为队员没有设置位置信息或可用时间，需要完善个人资料。

### 6.5 交易问题

**Q: 如何修改已发布的商品信息？**
A: 进入战队出售管理页面，找到商品，点击「编辑」按钮进行修改。

**Q: 商品出售后如何下架？**
A: 进入战队出售管理页面，找到已出售的商品，点击「下架」按钮。
